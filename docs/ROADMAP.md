## 1. 產品願景

**目標使用者**：日語零基礎～N5 程度的自學者（觀光旅遊、留學準備、興趣）。

**學習路徑**（App 導引順序）：

```
假名（v1 已完成）→ 主題單字 → 情境句子 → 基礎文法 → N5 檢定模擬
```

**核心原則（不變）**：
- 沿用 2c 設計系統（`lib/core/theme/app_theme.dart`）：靛藍/金黃/8px/硬陰影
- 沿用學習引擎：熟練度 0..5 加權出題、錯題本、統計
- 所有內容資料 = 純 Dart const 表（單一事實來源、可測試、離線可用）
- 每個里程碑保持 `dart analyze` 零 issue + 測試全綠才合併

## 3. M1 — 主題單字練習（本次實作）

### 功能需求

1. **單字庫**：105 詞（7 主題 × 15），全部 N5 程度
   - 主題：旅遊、交通、餐飲、購物、時間、日常、職場
2. **練習模式**：日→中 4 選 1
   - 題目卡顯示日文（漢字或假名）+ 讀音（假名，金黃小字）
   - 選項 = 4 個中文意思（同主題干擾優先，不足從全庫補）
   - 回饋規則與假名練習完全一致（綠✓/紅✕晃動/淡化/橫幅）
3. **範圍選擇**：首頁「單字練習」區 → 8 張卡（7 主題 + 全部單字）+ 錯題複習
4. **熟練度/統計**：共用現有 mastery map（key = `v_<日文>`）與全域統計
5. **單字錯題本**：獨立儲存（`vocab_wrong`），錯題頁改雙 tab（假名/單字），各自可重練/清除

### 資料結構

```dart
enum VocabTopic { travel, transport, food, shopping, time, daily, work }

class VocabWord {
  final String jp;      // 顯示字（漢字或假名），全庫唯一，key = 'v_<jp>'
  final String reading; // 假名讀音
  final String zh;      // 中文意思（繁體）
  final VocabTopic topic;
  final int jlpt;       // 5
}

enum VocabPool { all, travel, ..., work, wrongReview } // 練習範圍
```

### 技術改動

- `QuizGenerator` 泛型化：`QuizGenerator<T>`（keyOf 取熟練度 key、valueOf 取選項文字），假名/單字共用加權出題與選項生成
- `WrongNotifier` 參數化 storage key → `wrongProvider`（假名）+ `vocabWrongProvider`（單字）
- 練習頁共用 widget 抽出：`PracticeHeader`、`OptionButton`、`FeedbackBanner`

### 驗收標準

- [ ] 105 詞、jp 唯一、每主題 15 詞（測試保證）
- [ ] 單字 4 選 1 完整答題流程（widget test）
- [ ] 單字答錯進單字錯題本、複習答對遞減消化
- [ ] 假名功能零回歸（既有測試全綠）

## 4. M2 — 練習深度（單字）

1. **中→日反向**：題目中文、選項日文（附讀音）
2. **讀音輸入題型**：顯示漢字，輸入假名讀音或羅馬拼音（重用 AnswerChecker + alias 機制，拼音→假名轉換表）
3. **SRS 間隔複習**：mastery 增加 `nextDueAt`（熟練 0=立即、1=1天、2=3天、3=7天、4=14天、5=30天）；首頁顯示「今日待複習 N 個」
4. **每日目標**：設定每日題數目標（預設 20），首頁進度環，連續達標天數（真正的 streak days，取代目前的最佳連對展示位）

## 5. M3 — 情境句子

1. 5 情境 × 10 句（機場報到、電車購票、飯店入住、餐廳點餐、商店購物）
2. 資料：`SentenceItem { jp, reading, zh, blanks[], scene }`
3. 題型：
   - **克漏字**：句中挖一個詞，4 選 1（例：すみません、駅は＿＿ですか → どこ）
   - **重組**：打亂的詞塊按順序點選成句
4. 句子錯題本併入錯題頁第三 tab

## 6. M4 — 基礎文法

1. N5 文法 20 課：です/ます體、助詞（は・が・を・に・で・へ・と・も・から・まで）、疑問詞、て形、ない形、た形、形容詞變化
2. 資料：`GrammarPoint { title, explanation, examples[], quiz[] }`
3. 流程：卡片式教學（2-3 頁滑動）→ 4 選 1 練習 5 題 → 完成打勾
4. 首頁新增「文法」區，線性解鎖（前一課完成才開下一課）

## 7. M5 — 語音

1. `flutter_tts` 日文發音：單字卡/句子卡加 🔊 按鈕（設計稿本來就有，補上）
2. 聽力題型：播放讀音 → 4 選 1 選字或選義
3. 假名練習也加發音（點卡片聽音）

## 8. M6 — 檢定與同步

1. N5 模擬小測：文字語彙 20 題、計時 10 分鐘、交卷評分
2. 成績歷史頁（折線趨勢，dataviz）
3. Supabase 匿名帳號雲端備份（mastery/錯題/統計）
4. Google Play 上架準備：正式 keystore、隱私權政策、商店素材
